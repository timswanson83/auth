📋 Setup Guide — Azure App Registration
1

Register app in Azure (5 min, one-time)

Go to portal.azure.com → search "App registrations" → New registration.

Name: Biz Dev 2026 Account types: This organizational directory only Redirect URI: Single-page application (SPA) URI value: (paste the URL of this HTML file)

After registering, copy the Application (client) ID and Directory (tenant) ID into the fields above.

2

Grant SharePoint permissions

In your app registration → API permissions → Add → Microsoft Graph → Delegated:

Sites.ReadWrite.All Files.ReadWrite.All

Click Grant admin consent if you have admin rights. If not, ask your M365 admin — it's a one-click approval for them.

3

Sign in and test

Enter your Client ID and Tenant ID above → click Sign in with Microsoft → your normal M365 login window appears → click Test Connection to confirm.

4

Share with your team

Upload this HTML file to your SharePoint document library. Share the link with colleagues — they sign in with their own M365 account. Everyone reads and writes to the same live list.

Each team member signs in with their own Microsoft account — no shared passwords needed.
